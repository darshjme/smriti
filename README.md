## Overview

`agent-cache` implements **semantic and exact caching for llm calls — reduce costs 30%+ with zero-dependency tf-idf similarity matching and ttl-aware lru cache.**

Inspired by the Vedic principle of *स्मृति* (Smriti), this library brings the ancient wisdom of structured discipline to modern LLM agent engineering.

No external dependencies. Pure Python 3.8+. Drop it in anywhere.

## Installation

```bash
pip install agent-cache
```

Or clone directly:
```bash
git clone https://github.com/darshjme/agent-cache.git
cd agent-cache
pip install -e .
```
